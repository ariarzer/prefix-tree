# Префиксное дерево поиска

```js 
const data = {
    id1: // идентификатор сущности 
        [
            'text1',
            'test1',
            // список строк, по которым сущность можно найти
        ]
}
const tree = new prefixTree(data);

tree.find('text') // => отдаешь строку поиска, получаешь список индектификаторов сущностей, к которым она подходит
```
